# 1. 初步认识网页

# 网页是什么？

来看百度百科的定义：

  • 网页是构成网站的基本元素，是承载各种网站应用的平台。通俗地说，您的网站就是由网页组成的，如果您只有域名和虚拟主机而没有制作任何网页的话，您的客户仍旧无法访问您的网站。
  • 网页是一个包含HTML标签的纯文本文件，它可以存放在世界某个角落的某一台计算机中，是万维网中的一“页”，是超文本标记语言格式。网页通常用图像档来提供图画。网页要通过网页浏览器来阅读。

说人话就是：

  • 网页是个纯文本文件。
  • 要想发布一个网站，就一定要有网页。
  • 要用浏览器才能显示网页内容。

看到这里，也许你会有一些疑惑：为啥说网页是纯文本文件啊，网页里边不是有很多图片和视频吗？html标签是什么？别急，顺着往下看就能理解了。

# 写一个最简单的网页

  1. 新建一个文本文档，取名叫1.txt。打开这个文件，输入123，保存并关闭文件。
  2. 把文件后缀名从txt修改成html。

再次点开文件，发现是用浏览器打开的，并且可以在浏览器上看到刚才输入的内容123。这就是一个最简单的网页。

看到这你可能又会有点疑惑：这文件改个后缀咋就说它成了网页呢，它到底是个网页还是个文本文档啊？

我们能看到的网页内容，是浏览器逐行解析、理解了这个文件的内容后，按照规定的方式把内容展示出来。事实上这个文件的内容还是一开始的123，只是用更好看的方式将内容展示了出来而已。复杂的网页也是一样的，用文本编辑器来查看，可以发现它也只不过是一行行文本；**全靠浏览器去解读这些文本，才显示出了五花八门的内容。**网页里面的图片（或视频）事实上就是放了个链接，当浏览器解读网页知道了这是个图片，就会将这个链接所代表的图片显示出来。

**下面就来试试在刚才的网页中加入图片吧。**自己上网找张图片复制它的链接，然后添加到刚才的文本文件，保存退出，点开网页。

123
https://album.biliimg.com/bfs/new_dyn/dff0e5850a0b3e339caa6e113aad9e3b377713418.jpg

**是不是没有看到图片，只能看到这个链接本来的样子，也就是一堆字？那就对了。**之前浏览器为什么能显示123，是因为它解读了文件，发现了有123这么一句话，它就把这句话显示了出来。现在加多了一句话，那浏览器解读后，自然也把这句话显示出来了。

**我图片呢？它怎么不自动识别出这是图片链接然后显示成图片啊？**答案是浏览器还没有智能到这个地步；并且如果它真的自动把链接显示成图片了，但我原本是想让浏览器显示一串链接，你这不是捣乱吗？所以，想要让浏览器识别出这是个图片的链接，就要告诉它这是张图片，你浏览器要把这个链接给我解析成图片来显示咯。用什么方式来告诉浏览器呢？答案就是标签。

# 标签与属性

标签又可以叫做元素。

想要显示一张图片，那就用标签来告诉浏览器。

<img src="https://album.biliimg.com/bfs/new_dyn/dff0e5850a0b3e339caa6e113aad9e3b377713418.jpg">

上面这个就是一个标签，标签名是img，浏览器解析到img标签，就会知道，哦这是个图片而不是一堆字。

里边的这一大串，是这个img标签的src属性。属性是什么？属性就是物体的信息，用于描述物体的特征，例如一个人，它就会有身高、年龄等属性。属性包括属性名和属性值，例如这个人的身高是1米7，年龄是30岁。

那么图片有什么属性呢？图片最重要的属性就是图片的链接，然后是图片的宽、高等等。

除了图片标签，还有很多其他的标签。**标签其实就是物体，就像桌子，椅子，电脑、耳机……不过在网页里没有这些东西，有的是图片、视频、输入框、按钮、列表……**在网页中添加以下的标签，再用浏览器打开，可以发现网页中包括了几样东西：一个图片、一个按钮、一个输入框、一个列表。

<img src="https://album.biliimg.com/bfs/new_dyn/dff0e5850a0b3e339caa6e113aad9e3b377713418.jpg">
<button>按钮</button>
<input type="text">
<ul>
    <li>1</li>
    <li>2</li>
    <li>3</li>
    <li>4</li>
    <li>5</li>
</ul>

看上面的例子可以发现标签有两种写法：

  • 一种是像img、input这种的，<xxx>
  • 另一种是像button、ul、li这种的，是<xxx></xxx>

第二种写法为什么要用两部分呢？是因为这种标签里需要写一些内容。所以要用两部分包住

  • button标签里面写了“按钮”两个字，浏览器解析后的页面，按钮的提示字眼就是“按钮”俩字。
  • ul表示的是列表，li表示的是列表项。**ul里面的内容是5个li标签。**li里面的内容分别是1、2、3、4、5。标签里面是可以放其他标签的，但也要遵守一定的规则。假如电脑也是一个标签computer，那么电脑下面就包括了固态、内存、cpu等标签。他们是父子标签关系。

# 快速创建网页

  1. 在vs code中，创建一个文件，后缀用html。

  2. 切换到英文输入法，输入!，这时候会弹出提示，选择第一个，按下回车。发现会蹦出来这么一堆代码。这就是一个网页标准的基本结构。

    <!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Document</title>
    </head>
    <body>

    </body>
</html>

    第一行先不看，从第二行开始看。可以发现，网页包括了html标签，html标签下有2个子标签，分别是head和body。其中head标签下的内容，浏览器是不会把他们放到网页中显示给用户看的，head标签下还有几个子标签，包括meta和title。

    • meta标签用于给网页配置配置信息（官方说法叫元信息，其实就是这个网页的一些属性，例如作者是谁、关键词有哪些、页面描述是什么）。这里的两个meta标签起到的作用分别是指定编码方式、设置页面缩放的规则。
    • title标签指定浏览器标签页的标题。

    而body标签中的内容会被展示到页面上。

# 本章总结

经过初步的认识和实操可以知道，网页是由一个个标签组成的文本文件，由浏览器解析标签并展示网页内容。每个标签都有各自的属性，有些标签里面可以添加添加其他的标签或内容。

在本章还介绍了如何在vs code快速创建html模板。

后续章节不会介绍具体标签的使用，根据需求查阅官方文档即可。

# 2 网页布局

刚才创建的网页非常丑，没有任何布局可言，就是因为只写了页面有哪些标签，但没有给这些标签添加样式，那浏览器只能按照他默认的办法来给你排列内容。那怎么给网页添加样式呢？答案是编写css层叠样式表。举个例子：

img {
   width: 1500px;
}

上面这个就是最简单的一个样式。它的意思是：选择所有的img标签，让他们都要遵循：宽度为1500个像素。那部分代码写在哪呢？这就涉及到css引入方式。

# css引入方式

  • 行内样式表：直接在标签上添加style属性，然后给属性指定值。

    例如在img标签内添加 style="width: 1500px;"。因为是在一个标签中设置的，所以只能对那一个标签生效。不推荐这种做法。**html是用来编写网页的结构的，css是用来编写样式的，结构和样式写到一起的话，代码结构就没有那么清晰。**但有时候还是要在标签上直接设置style的，行内样式表的优先级比以下两种都要高。

  • 内部样式表：在head标签下，创建style标签。style标签下就直接写上面的例子的内容。

  • 外部样式表：编写css文件，上面例子的内容直接写css文件里面。在html文件的head标签下，创建link标签，引入css文件。

    <link rel="stylesheet" href="./01.css">

# 选择器

# 格式与含义
img {
   width: 1500px;
}

再来看一遍这个例子，上面对它的解释是：选择所有的img标签，让他们都要遵循：宽度为1500个像素。示例代码中的“img”事实上就是标签选择器，表示选择的就是所有img标签。

# 选择器种类

常见的选择器有：

  • 通配选择器【*】。选择所有标签
  • 标签选择器【标签名】。选择指定的标签
  • id选择器【#id值】。在html代码中，每个标签都可以指定id这个属性，相当于这个标签的身份证。用这个身份证就能找到对应的标签。用id选择器，就能给指定的标签设置样式。
  • 类选择器【.类名】。在html代码中，每个标签都可以指定class这个属性，相当于把标签分组，或者说打上tag。用类选择器，就能给这个类下的标签指定样式。一个标签可以属于多个类。

可以多个选择器一起使用：

  • 后代选择器【 （空格）】
  • 子选择器【>】
  • 并集选择器【,】
  • 兄弟选择器【~】
  • 伪类选择器【:】（例如button:hover，input:focus）

光说可能很难懂，下面来看例子：

<!-- 结构文件 -->
<img id="img1" src="https://album.biliimg.com/bfs/new_dyn/dff0e5850a0b3e339caa6e113aad9e3b377713418.jpg">
<button class="c1">按钮</button>
<input type="text">
<ul>
    <li id="id1">1</li>
    <li class="c1">2</li>
    <li>3</li>
    <li>
        <b>4</b>
    </li>
    <li>5</li>
</ul>

/* 样式文件 */
/* 通配选择器 */
* {
    /* 设置外边距和内边距 */
    margin: 0; 
    padding: 0;
}
/* 标签选择器 */
img {
    /* 设置图片宽度 */
    width: 500px;
}
/* id选择器 */
#id1 {
    /* 设置字体大小 */
    font-size: 40px;
}
/* 类选择器 */
.c1 {
    /* 设置字体颜色 */
    color: #bfa;
}
/* 后代选择器 */
ul b {
    font-size: 20px;
}
/* 子选择器 */
ul>li {
    color: #abf;
}
/* 并集选择器 */
b,
li {
    font-size: 30px;
}
/* 兄弟选择器 */
input~ul {
    height: 300px;
    background-color: #fba;
}
/* 伪类选择器，意思是当鼠标指针放到img标签上的时候对应的样式 */
img:hover {
    /* 设置鼠标指针样式 */
    cursor: pointer;
}
# 选择器优先级

当不同的选择器为同一个标签的同一个属性设置了不同的值，应该遵守哪一条规则呢？这就要看选择器的优先级了。

选择器的优先级需要计算权重。*的权重是0000，标签选择器是0001也就是个位，类选择器是0010也就是十位，id选择器是0100也就是百位，行内样式是1000是千位。如果是复合选择器，那么权重可以累加，但是不能进位，百位永远大于十位。如果在属性的值后面加上”!important“，则该样式权重变成无穷大。

在上面的例子中，id是id1的那个标签既符合id选择器【#id1】，也符合标签选择器【b,li】，这两个选择器都为它指定了字体大小。但因为id选择器的权重更高，所以最终字体大小取了40px。

# 块元素和行内元素

在上面的例子中，我们能看到图片、按钮、输入框都在同一行，只有宽度不够了的时候才会换到下一行去。但即使宽度很够的情况下，列表也还是会另起一行来显示。

这就是主要的两种元素类型：块元素和行内元素。

  • 块元素：高被内容撑开，宽度默认为所在容器的100%，宽、高、内外边距都可以设置。
  • 行内元素：宽高都被内容撑开，不能设置宽高，不独占一行

还有其他的一些元素类型，例如行内块，又能不独占一行，又能设置宽高，input和img都是行内块原速。

# 盒子模型

页面可以看作是一个个盒子摆放而成。每一个标签都可以理解为一个方形的盒子，盒子里面可以放东西，并且放的东西也可以看作是一个盒子。下面举b站搜索页来辅助理解。

先看b站搜索页本身的结构：从上至下可以分成三个区域，分别为顶栏、主体区域、底栏。

  • 顶栏又可以分为左边和右边的区域。左边和右边还可以再细分
  • 主体区域包括了搜索框、视频列表、页码三个主要部分。每个部分还可以再细分
  • 底栏略

接下来看看画出的示意图：

  • 最外层的黑色盒子表示整个页面，黑色盒子里装了3个小盒子。
  • 顶栏这个盒子里面还装了2个盒子；这两个盒子里面还可以再装其他盒子
  • 主体部分也分成几个盒子，每个盒子内还可以再装其他盒子
  • 底栏略

再来看看每个盒子的构成。在下图，紫色盒子和绿色盒子是父子关系；绿色盒子宽高均为600px；紫色的盒子宽高均为100px，边框厚度为10px，外边距为100px，内边距为50px。

  • 外边距margin。外边距也属于盒子的一部分。正因为紫色盒子有100px的外边距，才会和绿色盒子的边界有一定的距离。

  • 边框border

  • 内边距padding。内边距表示的是内容与边框之间的距离。可以看到内边距的颜色是和内容的颜色一致的。

  • 内容content。给盒子设置的宽和高默认情况下是设置给内容区的，这种模式叫做content-box。下图中，紫色的部分是内边距+内容区，内容区的宽高为100px，而内边距均为50px，所以紫色的部分宽和高都是150px。

    另一种模式叫做border-box，也就是说，给盒子设置的宽高是给border以内的区域设置的。

# 文档流

文档流就是指html文档中的标签被浏览器解析到页面后是怎么排布的。

默认的排布方向是纵向的。也就是说，如果A、B标签都是块元素，且当A标签写在B标签之上，那么当浏览器显示页面时，A标签就是在B的上面。

那怎么让标签横向排布呢？有个方法叫浮动，知道有这么东西就行，平时几乎不用。用得多的方法是flex布局。

这里主要介绍的是定位。定位用于让盒子在某个盒子内移动位置或者在屏幕中固定在某个位置。定位还可以设置叠放层次实现遮挡效果。定位写法为：position：xxx。然后使用left、right、top、bottom来移动盒子的位置。xxx可以是以下选项：

  • static。也就是默认的标准流。
  • relative。盒子的位置还在原来的地方，用left、top等属性来移动盒子的位置，是相对于盒子原来的位置。
  • absolute。盒子的位置被清理掉。用left、top等属性来移动盒子的位置，是相对于这个盒子的父元素中最近的、并且要有定位的一个元素。
  • fixed。盒子的位置被清理掉。用left、top等属性来移动盒子的位置，是相对于浏览器的可视区域。
  • sticky。先是relative，当移动到一定距离后变成fixed。

# flex

无敌好用，学会这个，网页布局基本就拿下了。当父元素display:flex时，子元素全都变成行内块。当父元素display:flex之后，元素内部可以看作有横竖两条轴。一条主轴，一条副轴。

给父元素用的属性：

  • flex-direction：设置主轴方向（row、row-reverse、column）
  • justify-content：设置子元素在父元素主轴的排列方式
    • center挤在中间
    • flex-start挤在起点
    • flex-end挤在终点
    • space-around平分剩余空间
    • space-between贴到两边后平分剩余空间
  • align-items：设置子元素在父元素副轴的排列方式（单行专用）
  • flex-wrap：是否允许子元素换行
  • align-content：设置子元素在父元素副轴的排列方式（多行专用）

给子元素用的属性：

  • flex：空间足够时是否拉长flex-grow、包括空间不足时是否缩短flex-shrink、未分配多余空间前占据的主轴空间flex-basis。flex-basis默认值为auto，即这个子元素本身的大小。如果取值为百分比，那就取父元素的百分比。如果父元素取决于子元素或没有值，那就跟auto一样。设置了flex：1之后，width就无效了，它就纯按占据的权重来分配宽度了。

flex布局常配合的单位有vw。而vw常配合max-width、min-width使用。感兴趣的还可以去了解响应式布局、媒体查询。

# 3 和网页互动

html负责编写页面结构，css负责编写页面样式，而和页面互动靠js来编写。

js部分要讲的话可以讲得很深，但本文的目的只是快速入门，目标是能用就行。既然是能用就行的地步，那只要简单了解js的数据类型，学会如何操作页面中的标签就行。

# 数据类型

JS是弱类型语言，变量的类型会在运行过程中自动确定。

基本类型可以分成：String、Number、Boolean、Null、Undefined。String是字符串，Number是数字，Boolean是true或false，Null表示空（有值但是值为空），Undefined的值就是undefined表示没有东西。

引用类型主要就是数组、对象、函数。这里就不细讲了。

# 操作标签

举个例子，我要实现一个功能：页面中有两个输入框和一个按钮，当在输入框输入数字之后点击按钮，求和并把结果显示在页面上。因为只是简单举例，就不考虑用户的奇奇怪怪的输入了。

编写代码，说人话大致是这么个流程：**告诉浏览器让他待命，当我点击按钮之后帮我执行以下的操作：分别获取两个输入框的值，然后求和，并在一个位置显示结果。**用这个例子来引入一些比较专业的术语：

  • 告诉浏览器让他待命，也就是让浏览器监听用户的操作，也可以说是注册事件。事件就是操作。
  • 这个例子中，浏览器监听的操作是什么呢？**是点击按钮。**当我鼠标放上去按钮的时候不会执行操作，当我鼠标离开按钮的时候也不会执行操作做，只有当我点击按钮的时候才会帮我执行操作。点击按钮，就是一个事件。当然，鼠标放上按钮也是事件，鼠标离开按钮也是事件。但此处只要求浏览器监听鼠标点击按钮这个事件。
  • 执行的后续的操作，可以说成执行一个回调函数。回调函数的内容，就是获取输入框的值，求和并显示。

看完上面的解释，对专业术语有了基本的认识，相信就能理解如何操作网页中的标签了：获取标签对象，为他的某个事件指定一个回调函数。当标签对象触发了这个事件时，就会执行回调函数。

  • 获取标签对象：getElementById（）、getElementByTagName（）、getElementByClassName（）、querySelector（）、querySelectorAll（）……
  • 事件类型：onclick（）、移动onmousemove（）、onmouseout（）、经过onmouseover（）、onfocus（）、onblur（）、onmouseup（）、onmousedown（）、onmouseenter（）……
  • 注册事件的方式：
    • btn.onclick = function(){}
    • addEventListner（事件类型没有on，事件处理函数，useCapture）

说太多没用，自己写一次就知道了。

<input type="text" id="input1">
<input type="text" id="input2">
<button id="button">求和</button>

<script type="text/javascript">
    // 创建变量，获取标签对象
    let button = document.querySelector('#button')
    let input1 = document.querySelector('#input1')
    let input2 = document.querySelector('#input2')

    // 为按钮注册事件，也就是告诉浏览器，当按钮被点击时，执行函数doClick的内容
    button.addEventListener('click', function doClick() {
        // 求和
        const res = Number(input1.value) + Number(input2.value)
        // 弹出提示框
        alert(res)
    })
</script>
